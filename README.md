# Analisis Konversi Langganan Berdasarkan Tingkat Belanja Pelanggan

Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis apakah tingkat belanja pelanggan berpengaruh terhadap keputusan pelanggan untuk berlangganan (subscription).

Analisis dilakukan menggunakan SQL Server untuk pengolahan dan perhitungan data, serta Power BI untuk visualisasi dan penyajian hasil analisis.
Proyek ini dibuat sebagai bagian dari portofolio Data Analyst tingkat junior.

Tujuan Analisis

Menjawab pertanyaan berikut:
Segmen pelanggan mana yang perlu diprioritaskan untuk meningkatkan konversi langganan berdasarkan perilaku pembelian?

Dataset

Sumber data: Shopping Behavior Dataset ([Kaggle](https://www.kaggle.com/datasets/zubairdhuddi/shopping-dataset/))
Jumlah data: 3.900 pelanggan

Kolom utama yang digunakan:

Purchase Amount
Previous Purchases
Subscription Status
Review Rating

Dataset digunakan untuk melihat pola perilaku belanja dan hubungannya dengan keputusan berlangganan.

Tools yang Digunakan

SQL Server (SSMS)
Digunakan untuk:

Menyiapkan dan membersihkan data
Membagi pelanggan ke dalam beberapa segmen berdasarkan tingkat belanja
Menghitung jumlah pelanggan, jumlah pelanggan berlangganan, dan tingkat konversi

Power BI
Digunakan untuk:

Menampilkan KPI utama
Visualisasi distribusi pelanggan per segmen
Visualisasi tingkat konversi langganan

Proses Analisis

Mengimpor dataset ke SQL Server
Melakukan pengolahan data menggunakan SQL
Membagi pelanggan ke dalam segmen Low, Medium, dan High berdasarkan tingkat belanja
Menghitung tingkat konversi langganan pada setiap segmen
Menyajikan hasil analisis dalam bentuk dashboard Power BI
Menarik kesimpulan berdasarkan hasil visualisasi
